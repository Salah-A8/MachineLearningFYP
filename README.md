# Machine Learning Sentiment Classifier (Final Year Project)

This project investigates whether gameplay, pricing, and engagement metrics
can be used to predict user sentiment for video games on Steam.

## Models Implemented
- Random Forest
- Decision Tree
- Logistic Regression
- Support Vector Machine (SVM)

## Techniques Used
- Feature selection
- SMOTE for class imbalance
- Accuracy, confusion matrix, ROC curves

## Dataset
The dataset used in this project is publicly available and can be downloaded from:
https://www.kaggle.com/datasets/antonkozyriev/game-recommendations-on-steam
